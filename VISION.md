# AquaClaw Vision 🦈

AquaClaw is a personal AI agent you run on your own machine. It answers you on the channels you already use, executes tasks autonomously, remembers things, and gets smarter the more you use it.

## Core principles

**1. Local-first.**
The gateway runs on your hardware. Your conversations stay on your machine. No mandatory cloud sync, no usage dashboards for someone else's analytics. Your data is yours.

**2. Any channel, any model.**
You already use WhatsApp, Telegram, Discord, or Slack. AquaClaw meets you there — you don't change your habits for the tool. Same for models: Claude, GPT, Gemini, or a local Ollama model you downloaded yesterday.

**3. Actually does things.**
The agent has real tools: a shell, the file system, a browser, web search, notifications, memory. When you ask it to do something, it does it — it doesn't just describe what you could do.

**4. Always-on.**
AquaClaw runs as a daemon. It's there when you wake up, when a cron job triggers, when a webhook fires, when a new email lands. You don't have to open an app.

**5. Voice is first-class.**
You should be able to walk into a room, say "hey shark" and start a conversation. AquaClaw should respond. Voice is not a demo — it's a primary interface.

**6. Open and extensible.**
Skills let you inject custom instructions. Tools let you add capabilities. Channels let you connect new platforms. Everything is a file, a JSON config, or a markdown document.

## What AquaClaw is not

- A hosted chatbot with a SaaS pricing page
- A coding IDE (use Claude Code for that)
- A replacement for human relationships
- Locked to one AI provider

## The shark metaphor

Sharks are ancient, efficient, and perfectly adapted. They don't overthink. They move through their environment with purpose. AquaClaw should feel like that: present, capable, unsentimental, fast.
